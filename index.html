kanji review

How does it work ?

This is a simple application designed to review and learn japanese kanji.

All kenji are from the Genki textbook. Each level includes 2 lessons.

Cards

You get a deck of cards showing the prononciation and you have to guess which kanji it is.

Markers

You can mark your cards to help you reviewing them. Use the markers as you like, they are there to help you.

  • Add to favorite
  • Mark as right
  • Mark as wrong
  • Remove item from deck

Buttons

  • ← Previous Show the previous card
  • Hint Show hint to help you guess the kanji
  • Answer Show the kanji
  • Next → Show the next card

Options

  • Go to first card Rewind the deck and show card number #1
  • Redraw cards Shuffle the current deck and show the first card
  • Reset Reset the application to it's initial state

Levels

The levels select box allows you to filter cards according to the levels you wish to review. Hold the CTRL (CMD on Mac) or SHIFT key to select multiple levels.

Show / Hide Cards

This option lets you show or hide cards marked with one of these markers : Favorite, right and wrong. Check those you want to see, others will be hidden.
